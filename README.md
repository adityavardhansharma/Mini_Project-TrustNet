## Run Locally

Clone the project

```bash
  git clonehttps://github.com/adityavardhansharma/Mini_Project-TrustNet.git
```

Go to the project directory

```bash
  cd client
```
Install Truffle Globaly

```bash
  npm install -g truffle
```

Install dependencies

```bash
  npm install
```
Deploy The Contracts

```bash
  truffle Compile
  truffle migrate
```

Run The Project

```bash
  npm start
```
